# 角色模型

角色只包含属性与属性值。

## Character 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | string | 角色 ID |
| `attributes` | object | 属性表 |

## Attribute 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `value` | number | 当前值 |
| `min` | number | 最小值 |
| `max` | number | 最大值 |

## 角色不保存的内容

| 内容 | 保存位置 |
| --- | --- |
| 标签 | 效果 |
| 剧情状态 | 事件 |
| 计数器 | 效果 |
| 已完成事件 | 对应事件字段 |
| 已解锁内容 | 对应效果或事件字段 |
